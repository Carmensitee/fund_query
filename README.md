# QDII LOF 基金限购查询

手机浏览器查看 38 只 QDII LOF 基金的申购状态和限购金额，电脑不开机也能用。

## 功能

- 38 只 QDII LOF 基金默认列表
- 申购状态查询：暂停申购 / 限制大额申购 / 开放申购
- 限购金额显示
- 按状态筛选（全部/暂停/限大额/开放）
- 数据变化检测：刷新后提示具体变动

## 使用

1. 手机浏览器打开部署地址
2. 输入密码
3. 查看基金列表，可切换筛选条件
4. 点「刷新数据」获取最新状态

## 部署

部署到 Streamlit Cloud，主文件为 `streamlit_app.py`，依赖 `streamlit` + `requests`。

## 数据来源

天天基金，实时抓取。
